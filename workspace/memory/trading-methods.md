# 交易方法定义

## 方法1：比尔威廉姆混沌理论
- 核心：鳄鱼线判断趋势 + 分形识别转折 + AO/AC判断动量
- 鳄鱼线：Jaw(SMMA13 shift8) + Teeth(SMMA8 shift5) + Lips(SMMA5 shift3)
- 分形：5根K线模式，中间最高/最低
- AO：5/34 SMA差值
- AC：AO - 5SMA(AO)

## 方法2：艾略特波浪理论
- 8浪循环：5浪推动 + 3浪调整
- 斐波那契比率：回撤0.382/0.5/0.618，延伸1.618/2.618
- 交替原则：2浪和4浪形态交替
- 通道：平行通道绘制
