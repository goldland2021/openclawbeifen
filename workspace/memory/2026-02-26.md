### 信号生成逻辑详细解释

#### 技术基础
1. **MA60移动平均线策略**
   - 基于60周期移动平均线
   - 计算价格与MA60的距离百分比
   - 分析趋势强度

2. **信号条件（放松条件版本）**
   ```
   min_distance = 0.1%  # 最小距离
   max_distance = 0.6%  # 最大距离
   trend_strength >= 60%  # 趋势强度阈值
   ```

3. **具体判断逻辑**
   - 距离条件：价格与MA60距离在0.1%-0.6%之间
   - 趋势条件：趋势强度≥60%
   - 方向判断：
     - 价格>MA60 → BUY信号
     - 价格<MA60 → SELL信号

#### 风险控制参数
- 止损：2%（距离入场价）
- 止盈：4%（距离入场价）
- 仓位：0.01手/笔（分批入场时）

#### 信号生成流程
```
1. 连接MT5 → 获取实时报价
2. 获取历史数据 → 70根5分钟K线
3. 计算MA60 → 基于60根K线
4. 计算距离 → (价格-MA60)/MA60*100
5. 计算趋势强度 → 最近5根K线在MA60上方的比例
6. 检查条件 → 距离+趋势条件
7. 生成信号 → BUY/SELL + 止损止盈
```

#### 系统架构
**混合方案（Python+MQL5）：**
- Python端：策略分析、信号生成
- MQL5端：交易执行、订单管理
- 文件通信：通过JSON文件传递信号

#### 信号文件格式
```json
{

  "take_profit": 5351.32, // 4%止盈

}
```
